# Дипломный проект: Chaos Organizer

Chaos Organizer — это веб-приложение для организации личного хаоса, аналог чата "Избранное" в Telegram. Приложение позволяет сохранять текстовые заметки, ссылки, координаты, медиафайлы и обычные файлы с возможностью синхронизации между различными устройствами.

## 🚀 Обязательные функции

1. **Сохранение в истории:** Приложение сохраняет все отправленные текстовые сообщения и медиафайлы.
2. **Кликабельные ссылки:** Все отправленные http/https ссылки автоматически парсятся и становятся кликабельными.
3. **Загрузка файлов:** Поддерживается загрузка изображений, видео, аудио и других файлов как через нажатие на иконку 📎 (скрепка), так и с помощью **Drag & Drop** (перетаскивание файлов прямо в окно браузера).
4. **Скачивание файлов:** При отправке обычного файла (не медиа), в чате формируется ссылка для его скачивания на компьютер пользователя.
5. **Ленивая подгрузка (Пагинация):** При открытии приложения загружаются только последние 10 сообщений. При прокрутке истории в самый верх автоматически подгружаются следующие 10 старых сообщений.

---

## ⭐ Дополнительные функции (Реализовано 5 штук)

1. **Синхронизация (WebSockets):** Если приложение открыто в нескольких окнах или вкладках одновременно, весь контент (сообщения и файлы) синхронизируется между ними в реальном времени.
2. **Отправка геолокации:** При нажатии на иконку 📍 приложение запрашивает координаты пользователя и отправляет в чат кликабельную ссылку на Яндекс.Карты.
3. **Воспроизведение медиа (API браузера):** Загруженные картинки, видео (`<video>`) и аудио (`<audio>`) отображаются и воспроизводятся прямо внутри чата.
4. **Поддержка оформления кода:** Текст, обернутый в тройные обратные кавычки ( ` ``` ` ), рендерится в чате как блок с кодом.
5. **Отправка команд боту:** В чате присутствует бот, который реагирует на 5 различных команд (отправляются в формате `@bot: <команда>`):
